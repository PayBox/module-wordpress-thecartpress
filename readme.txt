Platron payment gate for thecartpress

== Description ==

Russian payment gate Platron (www.platron.ru) now integrated to JigoShop. Platron include more than 20 payment system and have only one API and commerce offer. With this plugin integration will more easy!!!

== Installation ==

1. Upload plugin into wp-content/plugins
2. Activate and install the plugin Platron through the 'Plugins' menu in WordPress
3. Register on www.platron.ru
4. Get and paste settings through the 'TCP Settings->Payment Gates->Platron'
